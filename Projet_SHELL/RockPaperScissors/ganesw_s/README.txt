Bonjour !

Notre projet se découpe en deux codes : l’un “shifumi” classique demandé dans le sujet et “shifumi_with_bonus” où nous avons une forme du jeu un peu plus développée.

Shifumi_with_bonus : le programme se lance sans argument. A ce stade, nous avons deux modes de jeu : 
* Singleplayer.
* Multiplayer.

- Dans le singleplayer : 
  - 1) Normal : Demandé dans le sujet
  - 2) Duel : RPS contre l’ordinateur où la partie s’arrête dès que le PC ou le Joueur arrive au nombre de parties entrées. (LE VRAI SHIFUMI !)
Dès que l’une des deux parties gagne, nous avons soit la possibilité d’accéder au menu ou de quitter le jeu.

- Dans le multiplayer : 
  Il n’y a qu’un seul mode où deux joueurs entrent leurs noms, choissisent le nombre de parties qu’ils veulent jouer et chacun entre 1.2.3 en fonction de ses choix. Ce qu’ils entrent n'est pas affiché à l’écran.
